## ✨ المميزات الرئيسية

### 📚 المحتوى التعليمي
- **6 مستويات تعليمية** متدرجة (الأساسيات → الإتقان المهني)
- **24 وحدة دراسية** شاملة
- **80+ درس تفصيلي** مع أمثلة عملية
- **114+ سؤال تفاعلي** مع تصحيح فوري
- **500+ تمرين** مع حلول مفصلة
- **30+ دراسة حالة** من قطاعات متنوعة

### 🛠️ الأدوات التفاعلية
1. القيد المحاسبي التفاعلي
2. مُنشئ القوائم المالية
3. مُحلل النسب المالية (15+ نسبة)
4. حاسبة الإهلاك (3 طرق)
5. حاسبة الرواتب والضرائب
6. حاسبة نقطة التعادل
7. حاسبة القيمة الزمنية للنقود
8. حاسبة ضريبة القيمة المضافة (VAT)
9. محلل رأس المال العامل

- **300+ مصطلح** في معاجم متعددة
- **40+ معيار IFRS** مرجعي
- **35+ معيار تدقيق** دولي (ISA)
- **100+ نسبة مالية** موسعة
- النظام الضريبي السعودي الكامل
- دليل الحسابات السعودي
- دليل الموازنات التقديرية
- 20+ شركة نموذجية ببيانات مالية

### 📊 لوحة تحكم الطالب
- تتبع التقدم في رحلة التعلم
- إحصائيات شاملة (دروس مكتملة، اختبارات، وقت الدراسة)
- نظام إنجازات (Achievements) تفاعلي
- حفظ الدروس المفضلة والملاحظات

## 🎨 التقنيات المستخدمة

| التقنية | الاستخدام |
|---------|-----------|
| **Next.js 16** | إطار العمل الرئيسي |
| **TypeScript 5** | لغة البرمجة |
| **Tailwind CSS 4** | التصميم والتنسيق |
| **shadcn/ui** | مكونات الواجهة |
| **Zustand** | إدارة الحالة |
| **Cairo, Tajawal, Amiri** | خطوط عربية احترافية |

## 🚀 التشغيل محلياً

```bash
# تثبيت التبعيات
bun install

# تشغيل بيئة التطوير
bun run dev

# بناء المشروع للإنتاج
bun run build

# تشغيل النسخة المبنية
bun run start
```
